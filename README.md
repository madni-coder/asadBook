# My Business Spreadsheet App

A clean, mobile-responsive spreadsheet-like web app UI built with Vite, React, Tailwind CSS, and lucide-react icons.

## Features

-   Sidebar with app name, search, add register, and register list
-   Main section with blue header, tabs, and action buttons
-   Spreadsheet table with 5+ columns, 8 empty rows, and S.No. auto-numbering
-   Airtable/Google Sheets inspired design
-   Fully mobile responsive (collapsible sidebar, horizontal scroll, touch-friendly)

## Getting Started

1. Install dependencies:
    ```sh
    npm install
    ```
2. Start the development server:
    ```sh
    npm run dev
    ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

---

-   Built with [Vite](https://vitejs.dev/), [React](https://react.dev/), [Tailwind CSS](https://tailwindcss.com/), and [lucide-react](https://lucide.dev/).
